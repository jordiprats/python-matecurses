fraccions

Fraccions estil 1r ESO

Modes: mix - Les 4 seccions en un sol full (per defecte) combinades - Només operacions combinades (·, :, +, –, parèntesis) simplificacio - Simplifica fraccions grans fins a la irreductible fraccio_enter - Fracció d'un nombre enter fraccio_de_fraccio - Fracció de fracció d'un enter

Exemples: python matecursa.py fraccions python matecursa.py fraccions --mode combinades --pages 3 python matecursa.py fraccions --mode mix --pages 5 --file fraccions.pdf

Usage:

matecursa fraccions [OPTIONS]

Options:

Name Type Description Default
--file text fitxer de sortida fraccions.pdf
--pages integer nombre de fulls 5
--mode choice (combinades | simplificacio | fraccio_enter | fraccio_de_fraccio | mix) Tipus: combinades simplificacio
--help boolean Show this message and exit. False
